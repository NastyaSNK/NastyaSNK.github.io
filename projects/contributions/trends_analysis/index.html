Анализ трендов

С 2020 года я обучаюсь в магистратуре НИТУ МИСИС по программе “Наука о данных”. На учебной практике для Агенства инноваций города Москвы я с командой решала задачу по теме анализа трендов. В этом посте я расскажу не о всем проекте, а только о моем исследовании, выполненном в данном проекте.

Задача

Задача была поставлена в общем виде: разработать метод для выявления и оценки трендов. Можно было выбрать любую отрасль, тип тренда (например технологический), источники данных. Полная свобода)

\

Сбор и анализ инфо о задаче

Работу над проектом я начала с поиска ответов на следующие вопросы:
1️⃣Что такое тренд?
2️⃣Какие есть подходы к выявлению и оценке?

\

\

Методология

Я предложила решать задачу в 2 этапа:

1️⃣Выявляем тренды (прежде чем что-то оценить, нужно это обнаружить)
2️⃣Оцениваем тренды

\

Алгоритм вывления трендов

Для выявления трендов возможно использовать источники:
1️⃣Научные публикации
2️⃣Патенты
3️⃣Данные об инвестициях
4️⃣Запросы СМИ

Каждый из источников содержит информацию о потенциальном тренде.
Также возможно описать научную публикацию, деятельность компании, запросы в СМИ набором ключевых фраз. И если изначально сузить область поиска, то возможно получить пространство ключевых фраз, где одни будут ближе друг другу расположены, чем другие. А значит возможно из объединить. Объединенное “облако” ключевых фраз и будет характеризовать тренд. Схема работы алгоритма на рисунке ниже.

Два метода для оценки трендов

Помимо метода для выявления трендов мною были разработаны 2 метода для оценки трендов.

1️⃣Оценка трендов на основе матрицы коммерческого потенциала и исследовательской активности

Метод основан на показателе CAGR (compound annual growth rate) - совокупный среднегодовой темп роста, который показывает на сколько в среднем за год прирастает изучаемый параметр.
Показатель рассчитывается для каждого источника данных.
CAGR для каждого из источников данных в свою очередь объединяются с весами в показатели исследовательской активности и коммерческого потенциала.
В зависимость от расположения тренда в пространстве, которое характеризует исследовательскую активность и коммерческий потенциал, присвается стадия жизненного цикла: зарождение, рост, пик, зрелость,затухание.

Пример для этого метода оценки демонстрирует применение показателя CAGR только для 1го источника данных - Scopus. Немотря на это, даже использую только одну составляющую показателя рост и затухание тренда.

2️⃣Оценка трендов на устойчивость и прорыв

В этом методе рассчитываются покаказатели, характеризующие устойчивость и прорыв. После формируется список устойчивых и прорывных трендов по каждому из источников данных.
